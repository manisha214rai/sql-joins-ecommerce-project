# SQL Joins E-Commerce Project

## Overview

This project demonstrates SQL Joins using a real-world E-Commerce database. It covers join concepts from basic to advanced with practical business scenarios.

## Database Tables

### Customers

Stores customer information.

### Products

Stores product details and categories.

### Orders

Stores customer purchases and order information.

## SQL Concepts Covered

* INNER JOIN
* LEFT JOIN
* SELF JOIN
* CROSS JOIN
* Multi-Table JOINs
* JOIN + GROUP BY
* JOIN + HAVING
* JOIN + Subqueries
* Aggregation Queries
* Business Reporting Queries

## Practice Questions Included

* Customer and Product Analysis
* Order Reporting
* Customer Purchase Reports
* Product Sales Analysis
* Self Join Examples
* Cross Join Examples
* Advanced SQL Reporting


## Learning Outcome

This project helped me understand how SQL joins are used in real-world business scenarios such as customer analysis, sales reporting, and product performance tracking.
